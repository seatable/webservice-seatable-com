{{/* zoom: render an image inline; click it to open the same image full-size in a lightbox modal (Esc or click-outside to close). For hard-to-read screenshots/GIFs. {{< zoom image="images/foo.gif" alt="..." >}} {{< zoom image="images/foo.gif" alt="..." hint="Agrandir" >}} (optional label) `image` is a page-bundle path: images in a bundle are Page Resources, so we resolve via .Page.Resources.GetMatch -> .RelPermalink (a raw src="images/..." would 404), with a fallback to the raw path. Self-contained inline style+script, idempotent — mirrors the other custom shortcodes; distinct class/guard names so it coexists with the image-zoom hotspot lightbox after branch merges. */}} {{- $image := .Get "image" -}} {{- $alt := .Get "alt" | default "" -}} {{- $hint := .Get "hint" | default "Click to enlarge" -}} {{- $src := $image -}} {{- with .Page.Resources.GetMatch $image }}{{ $src = .RelPermalink }}{{ end -}}
