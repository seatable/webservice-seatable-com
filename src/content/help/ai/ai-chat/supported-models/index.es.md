El plugin de chat con IA en SeaTable es compatible con modelos de lenguaje de tres proveedores: **Anthropic**, **OpenAI** y **Mistral**. Usted elige qué modelo desea utilizar y aporta su propia clave API. Los costes por tokens se pagan directamente al proveedor correspondiente, no a SeaTable.

## Descripción general de los modelos

Cada proveedor ofrece dos modelos: uno rápido y económico para tareas sencillas y otro más potente para análisis exigentes.

| Proveedor | Modelo | Puntos fuertes | Tendencia de coste |
|---|---|---|---|
| **Anthropic** | Claude Haiku | Rápido, eficiente, ideal para consultas sencillas | económico |
| **Anthropic** | Claude Sonnet | Potente, preciso en análisis complejos | medio |
| **OpenAI** | GPT-4o mini | Rápido, rentable, versátil | económico |
| **OpenAI** | GPT-4o | Muy potente, alta calidad de respuesta | más elevado |
| **Mistral** | Mistral Small | Rápido, económico, buen rendimiento base | económico |
| **Mistral** | Mistral Large | Potente, ideal para tareas complejas | más elevado |

## Configurar la clave API

Para poder utilizar un modelo en el plugin de chat con IA, necesita una clave API del proveedor correspondiente. Esta se crea directamente en la página web del proveedor:

- **Anthropic**: Cree una clave API en [console.anthropic.com](https://console.anthropic.com/).
- **OpenAI**: Cree una clave API en [platform.openai.com](https://platform.openai.com/).
- **Mistral**: Cree una clave API en [v2.auth.mistral.ai/login](https://v2.auth.mistral.ai/login).

En los tres proveedores debe registrar un método de pago. Los costes dependen del número de tokens procesados y se facturan directamente por el proveedor.

## Recomendaciones según el caso de uso

### Consultas sencillas y respuestas rápidas

Para buscar valores individuales, resúmenes breves o preguntas de filtrado sencillas, los modelos económicos son los más adecuados: **Claude Haiku**, **GPT-4o mini** o **Mistral Small**. Estos ofrecen respuestas rápidas a bajo coste.

### Análisis complejos y evaluaciones de datos

Si desea evaluar conjuntos de datos extensos, identificar patrones o realizar cálculos de varios pasos, opte por **Claude Sonnet**, **GPT-4o** o **Mistral Large**. Estos modelos ofrecen resultados más precisos en tareas exigentes.

### Creación de textos y tareas de redacción

Para redactar correos electrónicos, informes o descripciones basándose en sus datos, se recomiendan **Claude Sonnet** o **GPT-4o**. Ambos modelos generan textos de alta calidad lingüística.

## Costes y facturación

Todos los costes por tokens se facturan directamente entre usted y el proveedor de IA correspondiente. SeaTable no cobra tarifas adicionales por el uso del plugin de chat con IA. Los precios exactos por token se encuentran en la página de precios del proveedor correspondiente.
