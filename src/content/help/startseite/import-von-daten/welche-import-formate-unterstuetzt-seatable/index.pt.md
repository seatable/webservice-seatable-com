SeaTable oferece-lhe uma vasta gama de opções para a recolha de dados: pode introduzir dados manualmente em tabelas e ter dados recolhidos através de [formulário web]({{< relref "help/base-editor/webformulare" >}}) ou utilizando automatismos.

SeaTable tem também uma poderosa função de importação de dados. Pode manusear praticamente qualquer **ficheiro Excel e CSV**. Além disso, é frequente que os utilizadores queiram migrar bases do **Airtable** para SeaTable. Através de um pequeno desvio, é também possível importar dados do **Microsoft Access** e do **Google Sheets**.

## Importação de dados com CSV

O CSV é um formato de intercâmbio de dados amplamente utilizado que é frequentemente o menor denominador comum entre dois produtos de software. Devido à sua estrutura simples, mantém o controlo total sobre os seus dados, mas pode importar um **máximo de uma tabela** ou base de cada vez.

[Saiba mais sobre a importação de dados com CSV aqui.]({{< relref "help/startseite/import-von-daten/daten-import-mithilfe-von-csv-in-seatable" >}})

## Importação de dados de Excel

Quando se trata de trabalhar com dados, o Excel não pode estar ausente, claro. SeaTable suporta a importação tanto do novo _formato de ficheiro_ _XLSX_ como do antigo _formato de ficheiro XLS_. Usando ficheiros Excel, pode também importar **várias tabelas** ou folhas de trabalho em simultâneo para uma base SeaTable. SeaTable converte cada folha de trabalho do Excel numa tabela.

[Saiba tudo sobre a importação de ficheiros Excel.]({{< relref "help/startseite/import-von-daten/import-von-excel-dateien-in-seatable" >}})

## Importação de dados do Airtable

Para a importação de bases do Airtable, SeaTable oferece tanto um caminho simples sem código como um script de migração mais flexível.

[Saiba aqui como migrar bases do Airtable para SeaTable.]({{< relref "help/startseite/import-von-daten/migration-von-airtable-bases-zu-seatable" >}})

## Importação de dados do Microsoft Access

Outra fonte a partir da qual se pode alimentar SeaTable com dados é o Microsoft Access.

[Saiba como importar dados do Microsoft Access.]({{< relref "help/startseite/import-von-daten/daten-import-aus-microsoft-access-in-seatable" >}})

## Importação de dados do Google Sheets

Também pode importar dados do Google Sheets para SeaTable.

[Saiba como importar dados do Google Sheets.]({{< relref "help/startseite/import-von-daten/daten-import-aus-google-sheets-in-seatable" >}})

## Importação de dados com DTABLE

As bases SeaTable podem ser exportadas em [formato de ficheiro DTABLE]({{< relref "help/startseite/import-von-daten/dtable-dateiformat" >}}) - e com tal ficheiro DTABLE pode, naturalmente, criar novamente uma base em SeaTable.

[Saiba mais sobre a importação de dados com DTABLE.]({{< relref "help/startseite/import-von-daten/erstellen-einer-base-aus-einer-dtable-datei" >}})
