Um jederzeit Anpassungen an bereits erstellten Automationen vornehmen zu können, haben Sie die Möglichkeit, Ihre angelegten Automatisierungregeln zu verwalten, zu gruppieren und zu verschieben.

## Automatisierungsregeln verwalten

1. Öffnen Sie eine **Base**, in der Sie bereits eine Automation angelegt haben.
2. Klicken Sie im Base-Header auf {{< seatable-icon icon="dtable-icon-rule" >}} und im Anschluss auf **Automatisierungsregeln**.
3. Fahren Sie mit dem Mauszeiger über die entsprechende Automatisierungsregel und klicken Sie auf die **drei Punkte**.
4. Nehmen Sie die gewünschten **Anpassungen** an der Automation vor.

![Automationsregeln verwalten](images/manage-automation-rules.jpg)

Folgende Verwaltungsmöglichkeiten stehen Ihnen für jede Automatisierungsregel zur Verfügung:

- **Regel umbenennen**
- **Regel duplizieren**
- [Log öffnen]({{< relref "help/base-editor/automationen/ausfuehrungslog-einer-automation-anzeigen" >}})
- [Regel pausieren]({{< relref "help/base-editor/automationen/automationen-stoppen" >}})
- [Regel löschen]({{< relref "help/base-editor/automationen/automationen-loeschen" >}})

Mehr zu den einzelnen Optionen finden Sie in den verlinkten Artikeln.

## Automatisierungsregeln gruppieren

Um Automatisierungsregeln in Gruppen zusammenzufassen, können Sie entsprechende Ordner anlegen.

Klicken Sie dazu auf **Regel oder Ordner hinzufügen** und dann auf **Ordner**.

![Ordner für Automatisierungsregeln anlegen](images/create-folder-for-automation-rules.jpg)

Geben Sie dem Ordner einen **Namen** und bestätigen Sie mit der **Enter-Taste**.

Fahren Sie mit dem Mauszeiger über den Ordner und klicken Sie auf die **drei Punkte**, um den Ordner umzubenennen oder zu löschen. Sie können auch direkt im Ordner eine neue Automatisierungsregel hinzufügen.

![Verwaltungsoptionen eines Automationsordners](images/automation-folder-options.jpg)

## Automatisierungsregeln verschieben

Um die Reihenfolge der Automatisierungsregeln zu ändern oder sie in einen Ordner zu verschieben, halten Sie die linke Maustaste gedrückt und ziehen die Regel **per Drag-and-Drop** an die gewünschte Stelle. Genauso können Sie Ordner verschieben.

![Automatisierungsregeln verschieben](images/move-automation-rules-and-folders.gif)
