С помощью автоматизаций вам больше не придется вручную отправлять электронные письма выбранным пользователям, а можно сделать так, чтобы они отправлялись автоматически. Просто определите соответствующую автоматизацию для любого табличного представления.

## Отправка электронных писем с помощью автоматизации

![Отправка электронных писем с помощью автоматизации](images/how-to-use-automations-for-locking-rows-3.png)

1. Нажмите на {{< seatable-icon icon="dtable-icon-rule" >}} в заголовке Base, а затем на **Automation Rules**.
2. Нажмите кнопку **Добавить правило**.
3. **Назовите** автоматизацию и определите **таблицу** и **представление, в** которых она должна работать.
4. Определите **триггерное событие, которое запускает** автоматизацию.
5. Нажмите **Добавить действие** и выберите **Отправить электронное письмо** в качестве автоматического действия.
6. Подтвердите с помощью кнопки **Отправить**.

## Создание автоматизации

Сначала определите **триггерное событие** для автоматизации.

![Триггерные события в основном доступны для выбора](images/trigger-options-for-archivating-rows.png)

{{< warning  headline="Примечание" >}}

Автоматическое действие **Отправить письмо** доступно после всех триггерных событий:

- Записи соответствуют определенным условиям после редактирования
- Добавляется новая запись
- Периодическое срабатывание
- Периодическое срабатывание для записей, удовлетворяющих определенным условиям

{{< /warning >}}

Затем в качестве автоматического действия выберите **Отправить электронное письмо**.

![Добавление автоматического действия](images/add-an-action.png) ![Добавление автоматического действия](images/example-send-email-action.png)

На следующем шаге сначала выберите **учётную запись электронной почты**, с которой вы хотите отправлять письма. Чтобы иметь возможность использовать вашу учётную запись электронной почты для отправки, вы **должны** сначала **добавить** её как стороннего провайдера в вашу базу SeaTable. Подробную инструкцию по этому шагу вы найдёте [здесь]({{< relref "help/integrationen/integrationen-innerhalb-von-seatable/gmail-fuer-den-versand-von-e-mails-per-smtp-einrichten" >}}).

{{< warning  type="warning" headline="Важное замечание"  text="Если вы **не** добавили в вашу базу SeaTable ни одной учётной записи электронной почты как стороннего провайдера, вам также **не** будут показаны никакие учётные записи. Таким образом, добавление учётной записи электронной почты является важным **предварительным условием** для автоматизации." />}}

![Выбор учётной записи электронной почты для отправки](images/select-email-account.png)

После выбора учётной записи электронной почты задайте **тему**, **получателя** и **содержание** письма. Вы можете ввести **несколько адресов электронной почты**, разделив их запятыми, а также в CC. Кроме того, у вас есть возможность добавить **вложения** из [столбца файлов]({{< relref "help/base-editor/spaltentypen/die-datei-spalte" >}}).

![Определение письма, отправляемого автоматизацией](images/define-email.png)

В завершение подтвердите автоматизацию нажатием кнопки **Отправить**.

![Подтверждение автоматизации](images/confirm-the-automation.jpg)

## Пример использования

Конкретный **случай использования** для такого типа автоматизации может возникнуть, например, если вы работаете в совете трудового коллектива компании. Вы хотите автоматически уведомлять отдел кадров о том, что вы одобрили увольнение сотрудника.

С вашей стороны это можно реализовать с помощью таблицы, в которой ведутся различные **данные** всех сотрудников вашей компании. В этом контексте вы могли бы фиксировать, среди прочего, **имена** сотрудников, их **статус занятости** и статус их **выплат заработной платы**.

![Пример таблицы конкретного случая использования](images/example-table-email-automation.png)

С помощью автоматизации SeaTable должен теперь автоматически отправлять **электронное письмо** в отдел кадров, как только вы одобрите увольнение сотрудника в совете трудового коллектива.

### Создание автоматизации

Сначала задайте автоматизации **имя** и выберите как **таблицу** (employee list), так и **представление** (employees ressort marketing), в которых должна действовать автоматизация.

![Базовые настройки автоматизации](images/basic-settings-automation.png)

В качестве **триггерного события** автоматизации выберите вариант **Записи удовлетворяют определённым условиям после обработки**. Чтобы письма отправлялись **исключительно** в случае увольнения сотрудника, добавьте в качестве **условия фильтра**, что запись в столбце **status of employment** должна быть изменена на **terminated**.

![Определение триггерного события](images/condition-of-the-trigger-event.png)

В качестве **автоматического действия** определите действие **Отправить электронное письмо**.

![Добавление автоматического действия](images/example-send-email-action.png)

На следующем шаге выберите **учётную запись электронной почты**, с которой вы хотите отправлять письма в отдел кадров. Чтобы иметь возможность использовать вашу учётную запись электронной почты для отправки, вы **должны** сначала **добавить** её как стороннего провайдера в вашу базу SeaTable. Подробную инструкцию по этому шагу вы найдёте [здесь]({{< relref "help/integrationen/integrationen-innerhalb-von-seatable/gmail-fuer-den-versand-von-e-mails-per-smtp-einrichten" >}}).

{{< warning  type="warning" headline="Важное замечание"  text="Если вы **не** добавили в вашу базу SeaTable ни одной учётной записи электронной почты как стороннего провайдера, на этом шаге вам также **не** будут показаны никакие учётные записи. Таким образом, добавление учётной записи электронной почты является важным **предварительным условием** для автоматизации." />}}

![Выбор учётной записи электронной почты для отправки](images/select-email-account-1.png)

После выбора вашей учётной записи электронной почты задайте **тему**, **получателя** и **содержание** писем. Вы можете ввести **несколько адресов электронной почты**, разделив их запятыми, а также в CC. В завершение у вас также есть возможность добавить **вложения** из [столбца файлов]({{< relref "help/base-editor/spaltentypen/die-datei-spalte" >}}).

{{< warning  headline="Совет"  text="С помощью фигурных скобок вы можете цитировать **содержимое столбцов таблицы** в тексте письма. Используйте эту функцию, чтобы автоматически вставлять **имена** уволенных сотрудников в текст." />}}

![Определение письма для автоматизации в случае использования](images/definition-email-example.png)

Когда вы закончите подготовку отправляемого письма, подтвердите автоматизацию нажатием **Отправить**.

### Тестирование автоматизации

Если после этого в выбранной таблице вы измените статус занятости сотрудника на **terminated**, заранее подготовленное **электронное письмо** будет автоматически отправлено в отдел кадров, чтобы уведомить их об увольнении.

![После изменения статуса занятости сотрудника на "terminated" заранее подготовленное письмо автоматически отправляется выбранным получателям ](images/test-automation-email.jpg)
