In SeaTable können Sie für Ihre Tabellen verschiedene Ansichten erstellen und damit dieselben Daten auf unterschiedliche Weise betrachten. Möchten Sie beispielsweise in einer Tabelle nach einem bestimmten Wert filtern, können Sie die gefilterte Ansicht der Tabelle speichern und somit in Zukunft schneller auf diese zugreifen, ohne erneut filtern zu müssen. 

## So erstellen Sie eine neue Ansicht in SeaTable

![Anlegen einer neuen Ansicht](images/create-table-view.jpg)

1. Klicken Sie auf den **Namen der aktuellen Ansicht**.
2. Klicken Sie auf **Ansicht oder Ordner hinzufügen** und wählen Sie den gewünschten **Ansichtstyp** aus.

![Eine Ansicht benennen](images/name-table-view.jpg)

3. Geben Sie der neuen Ansicht einen **Namen**.
4. Aktivieren Sie den Regler, falls die neue Ansicht nicht für alle sichtbar, sondern **privat** sein soll.
5. Bestätigen Sie mit **Abschicken**.

Die neu erstellte **Ansicht** öffnet sich im Anschluss automatisch und Sie können in dieser beispielsweise nach beliebigen Werten **filtern**, **sortieren** oder **gruppieren**.
