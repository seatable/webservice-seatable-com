По умолчанию SeaTable отображает все строки в табличных представлениях с компактной высотой строки, чтобы показать как можно больше записей. Однако при такой **простой высоте строки**, помимо прочего, обрезаются колонки [текста]({{< relref "help/base-editor/spaltentypen/die-spalten-text-und-formatierter-text" >}}), [ссылка]({{< relref "help/base-editor/tabellen/wie-man-tabellen-in-seatable-miteinander-verknuepft" >}}) и [множественным выбором]({{< relref "help/base-editor/spaltentypen/die-mehrfachauswahl-spalte" >}}), а также отображаются только маленькие **эскизы изображений**.

![Усеченное содержимое ячеек](images/small-row-height-cut-cells.png)

Такая высота строки может подойти для большинства случаев использования, но могут быть ситуации, когда вы хотите видеть больше содержимого соответствующих строк. Именно поэтому SeaTable позволяет настраивать высоту строк в табличных представлениях.

Для этого доступны четыре варианта высоты линии:

- Простой (значение по умолчанию для новых табличных представлений)
- Двойной
- Тройной
- Четырехсторонний

## Отрегулируйте высоту строки

![Отрегулируйте высоту ряда](images/set-row-height-new.gif)

1. Нажмите на значок **высоты строки** в опциях в табличном представлении.
2. **Выберите** нужную высоту ряда.

{{< warning  headline="Одна высота строки для всех строк"  text="Высота строки всегда настраивается для **всех строк** в табличном представлении. Высота отдельных рядов в представлении невозможна." />}}
