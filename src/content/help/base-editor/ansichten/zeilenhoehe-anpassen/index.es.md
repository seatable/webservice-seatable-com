Por defecto, SeaTable muestra todas las filas en las vistas de tabla con una altura de fila compacta para mostrar el mayor número posible de registros. Sin embargo, con esta **altura de fila simple**, entre otras cosas, se cortan las columnas de [texto]({{< relref "help/base-editor/spaltentypen/die-spalten-text-und-formatierter-text" >}}), [enlaces]({{< relref "help/base-editor/tabellen/wie-man-tabellen-in-seatable-miteinander-verknuepft" >}}) y [selección múltiple]({{< relref "help/base-editor/spaltentypen/die-mehrfachauswahl-spalte" >}}) y solo se muestran **miniaturas de imágenes**.

![Contenido de la celda truncado](images/small-row-height-cut-cells.png)

Esta altura de fila puede ser adecuada para la mayoría de sus casos de uso, pero puede haber situaciones en las que desee ver más contenido de las filas respectivas. Por eso SeaTable le permite ajustar la altura de las filas en las vistas de tabla.

Para ello se dispone de cuatro alturas de fila diferentes:

- Simple (el valor por defecto para las nuevas vistas de tabla)
- Doble
- Triple
- Cuádruple

## Ajustar la altura de la fila

![Ajustar la altura de la fila](images/set-row-height-new.gif)

1. Haga clic en el icono de **altura de fila** en las opciones de vista encima de la tabla.
2. **Seleccione** la altura de fila deseada.

{{< warning  headline="Una altura de fila para todas las filas"  text="La altura de la fila se ajusta siempre para **todas las filas** de una vista de tabla. No es posible realizar alturas de fila individuales dentro de una vista." />}}
