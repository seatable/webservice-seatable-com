In einer App können Sie beliebig viele **Seiten**, **Ordner** und **Links** erstellen, um die Daten in einer Base für eine bestimmte Nutzergruppe zugänglich zu machen, zu ordnen und ansprechend aufzubereiten.

{{< warning  type="warning" headline="Berechtigung in der Gruppe erforderlich"  text="Um eine App bearbeiten zu können, müssen Sie **Eigentümer** oder **Administrator der Gruppe** sein, in der sich die zugrundeliegende **Base** befindet. Einfache **Gruppenmitglieder** können die Apps lediglich ansehen und nutzen." />}}

## Seiten in Apps verwalten

Es gibt eine Vielzahl an [Seitentypen]({{< relref "help/app-builder/seitentypen-in-universellen-apps/ueberblick" >}}), die Sie schon von anderen Stellen in SeaTable kennen und mit nur wenigen Klicks in Ihrer App anlegen können.

### Eine neue Seite in einer App anlegen

1. Öffnen Sie eine **Base**, der Sie bereits eine App hinzugefügt haben.
2. Klicken Sie im Base-Header auf **Apps**.
   ![Klicken Sie im Base-Header auf Apps](images/click-apps-in-the-base-header.jpg)
3. Fahren Sie mit der Maus über die App und klicken Sie auf das **Stift-Symbol** {{< seatable-icon icon="dtable-icon-rename" >}}.
   ![Apps bearbeiten](images/Apps-bearbeiten.png)
4. Klicken Sie oben links auf **den orangen Kreis mit dem Plus-Symbol** und wählen Sie im Anschluss **Seite hinzufügen** aus.
   ![Add app page](images/Add-app-page.png)
5. Wählen Sie einen der [Seitentypen]({{< relref "help/app-builder/seitentypen-in-universellen-apps/ueberblick" >}}) aus und klicken Sie auf **Weiter**.
   ![Add app page and select page type](images/Add-app-page-and-select-page-type.png)
6. Geben Sie der Seite einen **Namen**, definieren Sie die zugrundeliegende **Tabelle** und legen Sie optional ein **Icon** für die Seite fest.
   ![Add app page and fill in details](images/Add-app-page-and-fill-in-details.png) 
7. Bestätigen Sie mit **Abschicken**.

### Namen und Icon einer Seite bearbeiten

Oft muss es beim Anlegen neuer Seiten schnell gehen und man wählt spontan irgendeinen Namen, den man später wieder ändern möchte. Deshalb können Sie über die **drei Punkte** jederzeit **Seiten umbenennen** und auf diesem Weg auch das **Icon** der Seite anpassen.

![Modify name and icon of a page in SeaTable Apps](images/Modify-name-and-icon-of-a-page-in-Universal-Apps.png)

### Seite duplizieren

Die Konfiguration von Seiten im SeaTable App Builder kann – insbesondere bei individuellen Seiten wie Dashboards – viel Zeit in Anspruch nehmen. Für den Fall, dass Sie bereits Seiten erstellt haben, die Sie nur leicht abwandeln möchten, bietet der SeaTable App Builder eine Funktion, mit der Sie viel Zeit und Aufwand sparen: Klicken Sie auf die **drei Punkte** und wählen Sie **Seite duplizieren**. Die Kopie übernimmt alle Inhalte, Einstellungen und Berechtigungen eins zu eins von der ursprünglichen Seite.

![Duplicate page in SeaTable App](images/Duplicate-page-in-Universal-App.png)

### Seite löschen

Nicht mehr benötigte Seiten können Sie jederzeit aus Ihrer App **löschen**. Beachten Sie dabei, dass die Löschung endgültig ist und gelöschte Seiten **nicht** wiederhergestellt werden können. Die **Daten** bleiben aber natürlich in der zugrundeliegenden Base **gespeichert**.

![Delete page in SeaTable App](images/Delete-page-in-Universal-App.png)

### Seite verschieben

Wenn Sie bereits einen **Ordner** in Ihrer App angelegt haben, können Sie Ihre Seiten über die **drei Punkte** in diesen verschieben.

![Move page to folder in SeaTable Apps](images/Move-page-to-folder-in-Universal-Apps.png)

Alternativ können Sie Seiten auch **per Drag-and-Drop** verschieben. Halten Sie dazu die Maustaste auf den **sechs Punkten** vor dem Seiten-Icon gedrückt, ziehen Sie die Seite in der Navigation an die gewünschte Stelle und lassen Sie los.

![Seiten einer App per Drag-and-drop verschieben](images/move-pages-via-drag-and-drop.gif)

## Ordner in Apps verwalten

Bei einer großen Anzahl an Seiten und Links sind **Ordner** hilfreich, um die Seiten und Links thematisch zusammenzufassen und die Navigation für die Benutzer übersichtlicher zu gestalten.

### Einen neuen Ordner in einer App anlegen

1. Öffnen Sie eine **Base**, der Sie bereits eine App hinzugefügt haben.
2. Klicken Sie im Base-Header auf **Apps**.
   ![Klicken Sie im Base-Header auf Apps](images/click-apps-in-the-base-header.jpg)
3. Fahren Sie mit der Maus über die App und klicken Sie auf das **Stift-Symbol** {{< seatable-icon icon="dtable-icon-rename" >}}.
   ![Apps bearbeiten](images/Apps-bearbeiten.png)
4. Klicken Sie oben links auf **den orangen Kreis mit dem Plus-Symbol** und wählen Sie im Anschluss **Ordner hinzufügen** aus.
   ![Add folder to SeaTable App](images/Add-folder-in-Universal-Apps.png)
5. **Benennen** Sie den Ordner und wählen Sie optional ein passendes **Icon** für diesen aus.
   ![Add new folder to app](images/Add-new-folder-to-app.png)
6. Bestätigen Sie mit **Abschicken**.

### Seite zum Ordner hinzufügen

Wenn Sie bereits einen Ordner in Ihrer App angelegt haben, können Sie in diesem auf die **drei Punkte** klicken und direkt eine **Seite hinzufügen**.

![Hinzufügen von Seiten zu einem Ordner in der App](images/add-page-via-folder-to-universal-app.png)

Auf dem gleichen Weg können Sie auch einen **externen Link** zum Ordner hinzufügen.

### Namen und Icon eines Ordners bearbeiten

Über die **drei Punkte** können Sie Ordner in Ihrer App jederzeit **umbenennen**. Zudem können Sie auf diesem Weg auch das **Icon** Ihres Ordners anpassen.

![Anpassung von Name und Icon eines Ordners in Ihrer App](images/modify-name-and-icon-of-a-folder-in-the-universal-app-new.png)

### Ordner löschen

Nicht mehr benötigte Ordner können Sie jederzeit aus Ihrer App **löschen**. Beachten Sie dabei, dass **alle Seiten**, die sich im Ordner befinden, ebenfalls **gelöscht** werden. Die Löschung ist endgültig. Das heißt: Sowohl der Ordner als auch die einzelnen Seiten können im Anschluss **nicht** wiederhergestellt werden.

![Löschen eines Ordners in Ihrer App](images/delete-folder-in-your-universal-app-new.png)

### Ordner verschieben

Ordner können Sie nur **per Drag-and-Drop** verschieben. Halten Sie dazu die Maustaste auf den **sechs Punkten** vor dem Ordner-Icon gedrückt, ziehen Sie den Ordner in der Navigation an die gewünschte Stelle und lassen Sie los.

![Ordner in der App per Drag-and-Drop verschieben](images/move-folders-in-the-universal-app-via-drag-and-drop.gif)

## Externe Links in Apps verwalten

Gelegentlich kommt es vor, dass Sie auf Ressourcen verweisen möchten, die nicht in SeaTable gespeichert sind. Dann setzen Sie einfach einen **externen Link** in der Navigation, damit die App-Nutzer per Mausklick die entsprechende Webseite öffnen können.

### Einen externen Link in einer App anlegen

1. Öffnen Sie eine **Base**, der Sie bereits eine App hinzugefügt haben.
2. Klicken Sie im Base-Header auf **Apps**.
   ![Klicken Sie im Base-Header auf Apps](images/click-apps-in-the-base-header.jpg)
3. Fahren Sie mit der Maus über die App und klicken Sie auf das **Stift-Symbol** {{< seatable-icon icon="dtable-icon-rename" >}}.
   ![Apps bearbeiten](images/Apps-bearbeiten.png)
4. Klicken Sie oben links auf **den orangen Kreis mit dem Plus-Symbol** und wählen Sie im Anschluss **Externen Link hinzufügen** aus.
   ![Add external link in app](images/add-external-link-to-apps.png)
5. Geben Sie die **Link-Adresse** und einen **Titel** ein.
   ![Add external link in app and fill in details](images/add-external-link-in-app.png) 
6. Bestätigen Sie mit **Abschicken**. Wenn Sie anschließend auf den Link in der Navigation klicken, öffnet sich die gewünschte Webseite.

### Adresse und Titel eines Links bearbeiten

Über die **drei Punkte** können Sie jederzeit den Link **umbenennen** und die **Link-Adresse anpassen**.

![Modify name and address of a link in SeaTable Apps](images/modify-name-and-address-of-an-app-link.png)

### Link löschen

Nicht mehr benötigte Links können Sie jederzeit aus Ihrer App **löschen**. Beachten Sie dabei, dass die Löschung endgültig ist und gelöschte Links **nicht** wiederhergestellt werden können.

![Delete a link in SeaTable App](images/delete-app-link.png)

### Links verschieben

Wenn Sie bereits einen **Ordner** in Ihrer App angelegt haben, können Sie Links über die **drei Punkte** in diesen verschieben.

![Move link to folder in SeaTable Apps](images/move-link-to-folder-in-app.png)

Alternativ können Sie Links auch **per Drag-and-Drop** verschieben. Halten Sie dazu die Maustaste auf den **sechs Punkten** vor dem Link-Icon gedrückt, ziehen Sie den Link in der Navigation an die gewünschte Stelle und lassen Sie los.

![Link in einer App per Drag-and-drop verschieben](images/move-links-via-drag-and-drop.gif)
