In an app you can create as many **pages**, **folders** and **links** as you like to make the data in a base accessible to a specific user group, to organize it and to present it in an appealing way.

{{< warning  type="warning" headline="Authorization required in the group"  text="To be able to edit an app, you must be the **owner** or **administrator of the group** in which the underlying **base** is located. Simple **group members** can only view and use the apps." />}}

## Manage pages in apps

There are a variety of [page types]({{< relref "help/app-builder/seitentypen-in-universellen-apps/ueberblick" >}}) that you already know from other places in SeaTable and can create in your app with just a few clicks.

### Create a new page in an app

1. Open a **base** to which you have already added an app.
2. Click **Apps** in the base header.
   ![Click Apps in the base header](images/click-apps-in-the-base-header.jpg)
3. Hover your mouse over the app and click the **pencil icon** {{< seatable-icon icon="dtable-icon-rename" >}}.
   ![Edit apps](images/Apps-bearbeiten.png)
4. Click on **the orange circle with the plus symbol** in the top left and then select **Add page**.
   ![Add app page](images/Add-app-page.png)
5. Select one of the [page types]({{< relref "help/app-builder/seitentypen-in-universellen-apps/ueberblick" >}}) and click **Next**.
   ![Add app page and select page type](images/Add-app-page-and-select-page-type.png)
6. Give the page a **name**, define the underlying **table** and optionally specify an **icon** for the page.
   ![Add app page and fill in details](images/Add-app-page-and-fill-in-details.png) 
7. Confirm with **Submit**.

### Edit the name and icon of a page

When creating new pages, it often has to be done quickly and you spontaneously choose a name that you want to change later. This is why you can **rename pages** at any time using the **three dots** and also change the page **icon** in this way.

![Modify name and icon of a page in SeaTable Apps](images/Modify-name-and-icon-of-a-page-in-Universal-Apps.png)

### Duplicate page

The configuration of pages in the SeaTable App Builder can take a lot of time – especially for individual pages such as dashboards. If you have already created pages that you only want to modify slightly, the SeaTable App Builder offers a function that will save you a lot of time and effort: Click on the **three dots** and select **Duplicate page**. The copy takes over all content, settings and authorizations one-to-one from the original page.

![Duplicate page in SeaTable App](images/Duplicate-page-in-Universal-App.png)

### Delete page

You can **delete** pages that are no longer required from your app at any time. Please note that the deletion is final and deleted pages **cannot** be restored. However, the **data** will of course remain **stored** in the underlying base.

![Delete page in SeaTable App](images/Delete-page-in-Universal-App.png)

### Move page

If you have already created a **folder** in your app, you can move your pages to it using the **three dots**.

![Move page to folder in SeaTable Apps](images/Move-page-to-folder-in-Universal-Apps.png)

Alternatively, you can also move pages **using drag-and-drop**. To do this, hold down the mouse button on the **six dots** in front of the page icon, drag the page to the desired position in the navigation and release.

![Move pages of an app via drag and drop](images/move-pages-via-drag-and-drop.gif)

## Manage folders in apps

With a large number of pages and links, **folders** are helpful to group pages and links thematically and make navigation clearer for users.

### Create a new folder in an app

1. Open a **base** to which you have already added an app.
2. Click **Apps** in the base header.
   ![Click Apps in the base header](images/click-apps-in-the-base-header.jpg)
3. Hover your mouse over the app and click the **pencil icon** {{< seatable-icon icon="dtable-icon-rename" >}}.
   ![Edit apps](images/Apps-bearbeiten.png)
4. Click on **the orange circle with the plus symbol** in the top left and then select **Add folder**.
   ![Add folder to SeaTable App](images/Add-folder-in-Universal-Apps.png)
5. **Name** the folder and optionally choose a suitable **icon** for it.
   ![Add new folder to app](images/Add-new-folder-to-app.png)
6. Confirm with **Submit**.

### Add page to folder

If you have already created a folder in your app, you can click on the **three dots** in this folder and **add** a **page** directly.

![Adding pages to a folder in the app](images/add-page-via-folder-to-universal-app.png)

You can also add an **external link** to the folder in the same way.

### Edit the name and icon of a folder

You can **rename** folders in your app at any time using the **three dots**. In addition, you can also customize the **icon** of your folder this way.

![Customizing the name and icon of a folder in your app](images/modify-name-and-icon-of-a-folder-in-the-universal-app-new.png)

### Delete folder

You can **delete** folders that are no longer required from your app at any time. Please note that **all pages** in the folder will also be **deleted**. The deletion is final. This means that neither the folder nor the individual pages can be restored afterwards.

![Deleting a folder in your app](images/delete-folder-in-your-universal-app-new.png)

### Move folder

You can only move folders **using drag-and-drop**. To do this, hold down the mouse button on the **six dots** in front of the folder icon, drag the folder to the desired location in the navigation and release.

![Drag and drop folders in the app](images/move-folders-in-the-universal-app-via-drag-and-drop.gif)

## Manage external links in apps

Occasionally you may want to refer to resources that are not stored in SeaTable. Simply add an **external link** in the navigation so that app users can open the corresponding website with a mouse click.

### Create an external link in an app

1. Open a **base** to which you have already added an app.
2. Click **Apps** in the base header.
   ![Click Apps in the base header](images/click-apps-in-the-base-header.jpg)
3. Hover your mouse over the app and click the **pencil icon** {{< seatable-icon icon="dtable-icon-rename" >}}.
   ![Edit apps](images/Apps-bearbeiten.png)
4. Click on **the orange circle with the plus symbol** in the top left and then select **Add external link**.
   ![Add external link in app](images/add-external-link-to-apps.png)
5. Enter the **link address** and a **title**.
   ![Add external link in app and fill in details](images/add-external-link-in-app.png) 
6. Confirm with **Submit**. When you then click on the link in the navigation, the desired website opens.

### Edit the address and title of a link

Using the **three dots** you can **rename** the link and **adjust the link address** at any time.

![Modify name and address of a link in SeaTable Apps](images/modify-name-and-address-of-an-app-link.png)

### Delete link

You can **delete** links that are no longer required from your app at any time. Please note that the deletion is final and deleted links **cannot** be restored.

![Delete a link in SeaTable App](images/delete-app-link.png)

### Move links

If you have already created a **folder** in your app, you can move links to it using the **three dots**.

![Move link to folder in SeaTable Apps](images/move-link-to-folder-in-app.png)

Alternatively, you can also move links **using drag-and-drop**. To do this, hold down the mouse button on the **six dots** in front of the link icon, drag the link to the desired position in the navigation and release.

![Move links in an app via drag and drop](images/move-links-via-drag-and-drop.gif)
